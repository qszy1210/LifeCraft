# LifeCraft

**LifeCraft** 是一个专注于收集和分享日常生活中实用物品和技巧的项目。我们相信“授人以鱼不如授人以渔”，因此我们不仅提供物品推荐，还会分享如何选择、使用和维护这些物品的技巧。

## 项目目标

- **收集**：整理和收集日常生活中常用的物品和工具。
- **分享**：分享如何选择、使用和维护这些物品的技巧。
- **学习**：帮助用户通过实践和学习，掌握生活中的实用技能。
